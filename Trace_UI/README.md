# TRACE System – Frontend (Skin Cancer Detection Platform)

Frontend for the TRACE skin cancer detection platform, built with React + Vite + Tailwind CSS.

## Features

- Authentication pages and dashboard flow
- New analysis workflow (upload -> processing -> results)
- Hair-removal preprocessing visualization
- Skin cancer classification result display
- YOLO localization overlay with confidence/coverage labels
- History and profile management pages

## Project Structure

- `src/components`: reusable UI components
- `src/pages`: route-level screens
- `src/services`: API/service layer
- `src/assets`: static assets
- `src/index.css`: global styles and Tailwind entry

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
